Usos principais do GIT no CMD:

1- git cloney
Copia a URL de projeto do GITHUB para a máquina

2- git add <nome do arquivo>.<extensão> ou git .
Colocar na minha stage área(Na minha máquina)

3- git commit -m "Apelido do DIR"
Me permite adcionar na minha máquina e me permite deixar pronto para postagens no github

4- git status
Analisa o status do meu projeto e como ele está em execução

5- git push
Me permite enviar o arquivo da máquina para o site do GitHub

6- git remote -v
Me permite analisar o meu push de usuário do git 

7- git pull
Me permite atualizar dados do git em meus arquivos de máquina

8- git fetch
Tb me permite Atualizar dados do git em meus arquivos de máquina

9- git branch
Me permite analisar qual branch do git estou usando (Padrão: main)

10- git branch <nome_do_branch>
Me permite criar um branch

11- git checkout <nome_do_branch_que_quer_acessar>
Me permite mudar e ir para outro branch do git

12- git diff . ou git diff <arquivo>.<extensão>
Me permite analisar o que foi alterado

13- git merge <branch_a_ser_copiado>
Me permite copiar arquivos de um branch para outro
